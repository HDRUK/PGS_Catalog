PGS Catalog Scoring Files

Each score file is formatted to be a gzipped tab-delimited text file, labelled by its PGS Catalog Score ID (e.g. PGS000001.txt.gz).

In the current catalog the scores have been extracted from the relevant publication and only been reformatted to have a consistent header (lines starting with #) listing relevant PGS Catalog information, e.g.: 

### PGS CATALOG SCORING FILE - see www.pgscatalog.org/downloads/#dl_ftp for additional information
## POLYGENIC SCORE (PGS) INFORMATION
# PGS ID = PGS idenfier, e.g. 'PGS000001'
# Reported Trait = trait, e.g. 'Breast Cancer'
# Original Genome Build = Genome build/assembly, e.g. 'GRCh38'
# Number of Variants = Number of variants listed in the PGS
## SOURCE INFORMATION
# PGP ID = PGS publication idenfier, e.g. 'PGP000001'
# Citation = Information about the publication
rsIDchr_namechr_positioneffect_allelereference_allele...

PGS Metadata File

Downloads of the full PGS Catalog metadata (data fields are described in more detail here), consists of an Excel file (.xlsx) containing all worksheets or an individual .csv file for each of the follwoing sheets:

Worksheet NameDescription
ReadmeInformation about the current PGS Catalog release.
ScoresLists and describes all PGS scores (excluding samples used for training, found in the Score Development Samples sheet).
Performance MetricsLists the performance metrics and sources for each PGS evaluation in the catalog (information about the samples that were evaluated is found in the Evaluation Sample Sets sheet).
Evaluation Sample SetsDescribes the samples used to evaluate PGS from the Performance Metrics sheet.
Score Development SamplesLists the source GWAS studies used to identify trait-associated variants, also describes the samples used in the GWAS studies and/or to train the PGS.
PublicationsLists information about the Publications present in the GWAS Catalog.
EFO TraitsLists the Traits that are present in the PGS Catalog according to the EFO Ontology.
